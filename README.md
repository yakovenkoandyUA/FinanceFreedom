## Build Setup:

```bash
# Download repository:
git clone https://github.com/yakovenkoandyUA/FinanceFreedom.git

# Go to the app:
cd FinanceFreedom

# Install dependencies:
npm install

# Server with hot reload at http://localhost:8081/
npm run dev

# Output will be at dist/ folder
npm run build
```
